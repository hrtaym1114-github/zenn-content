# Codex（クラウドコーディングエージェント）

## Codexとは

Codexは、OpenAIが提供するクラウドベースのソフトウェアエンジニアリングエージェントです。以前の「Codex」（2021年のコード補完モデル）とは全く異なる製品で、2025年に再定義されました。

主な特徴:
- **クラウドサンドボックス**で動作
- **並列タスク実行**が可能
- **GitHubリポジトリ**と連携
- タスク完了まで**1〜30分**
- 実行ログ・テスト出力の**引用による検証可能性**

## サンドボックス環境

Codexエージェントは**安全な隔離コンテナ**内で動作します。

- タスク実行中は**インターネットアクセスが無効**
- エージェントが触れるのはGitHubリポジトリから提供されたコードと、セットアップスクリプトで設定された依存関係のみ
- 外部Webサイト、API、その他サービスへのアクセス不可
- デフォルトコンテナイメージ「universal」には主要な言語・パッケージ・ツールがプリインストール

## できること

- 機能の実装（新規コード作成）
- コードベースに関する質問への回答
- バグ修正
- プルリクエストの提案
- テストハーネス・リンター・型チェッカーの実行
- ファイルの読み書き
- コマンドの実行

## セットアップスクリプト

Codexの環境をカスタマイズするには、リポジトリにセットアップスクリプトを配置します。これにより:

- 追加の依存関係をインストール
- 環境変数を設定
- ビルドツールを準備

## Playwright (Interactive) モード

GPT-5.4のCodex向け実験的スキルとして「Playwright (Interactive)」が追加されました。

- Webアプリケーションの**視覚的デバッグ**が可能
- Electronアプリにも対応
- 開発中のアプリを**ビルドしながらテスト**できる
- 従来のMCPプロトコル（約114,000トークン/タスク）に対し、CLIベース（約27,000トークン/タスク）で**75%のトークン削減**

## MCP（Model Context Protocol）対応

CodexはMCPサーバーとの接続をファーストクラスでサポートしています。

```toml
# codex config.toml
[mcp.playwright]
command = "npx"
args = ["@playwright/mcp@latest"]
```
