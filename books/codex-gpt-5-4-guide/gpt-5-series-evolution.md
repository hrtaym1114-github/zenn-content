# GPT-5シリーズの進化

```

  +-- GPT-5.3-Codex（2026年2月5日）
```

## GPT-5.2からの主な改善点

| 項目 | GPT-5.2 | GPT-5.4 | 改善幅 |
|------|---------|---------|--------|
| コンテキスト | 400,000トークン | 1,050,000トークン | **2.6倍** |
| 個別回答の正確性 | ベースライン | 33%改善 | - |
| 全体回答の正確性 | ベースライン | 18%改善 | - |
| OSWorld | 47.3% | 75.0% | **+27.7pp** |
| SWE-bench Pro | 55.6% | 57.7% | +2.1pp |
| Computer Use | なし | ネイティブ対応 | 新機能 |
| Tool Search | なし | 対応 | 新機能 |

## GPT-5.3-Codexとの関係

GPT-5.3-Codexはコーディングに特化したモデルとして2026年2月5日にリリースされました。GPT-5.4はこのコーディング能力を継承しつつ、汎用推論とComputer Useを追加した「統合モデル」です。

- **GPT-5.3-Codex**: コーディング最強だが汎用性に欠ける
- **GPT-5.4**: コーディング能力を維持しつつ汎用性を確保

実質的に、GPT-5.4はGPT-5.3-Codexの**上位互換**と位置づけられます。
