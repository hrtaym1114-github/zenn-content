# 料金体系

## API料金

| モデル | 入力 (1Mトークン) | 出力 (1Mトークン) |
|--------|-------------------|-------------------|
| GPT-5.4 | **$2.50** | **$15.00** |
| GPT-5.4 Pro | **$30.00** | **$180.00** |

## 割引・追加料金

| 条件 | 料率 |
|------|------|
| Batch + Flex | **半額** |
| Priority Processing | **2倍** |
| 長文コンテキスト（>272Kトークン入力） | 入力**2倍** + 出力**1.5倍** |
| Data Residency / Regional Processing | **+10%** |

:::message
**Batch + Flex**を活用すれば、リアルタイム性が不要なタスクを半額で処理できます。大量のタスクを一括処理する場合は積極的に活用しましょう。
:::

## コスト比較

GPT-5.4はGPT-5.2比でトークン効率が大幅に向上しているため、同じタスクをより少ないトークンで完了できます。結果として、単価は変わらなくても**実質コストは削減**される場合が多いです。

## Codex料金

Codexの利用料金は別体系です。ChatGPT Proプランに含まれるか、API経由で利用する場合はAPI料金が適用されます。
