# GPT-5.4とは

## 概要

GPT-5.4は、OpenAIが2026年3月5日にリリースした最新のフロンティアモデルです。ChatGPT、Codex、APIの3つのプラットフォームで利用できます。

GPT-5.4の最大の特徴は、以下の3つの能力を**単一モデル**に統合した点にあります。

1. **高度な推論能力**（GPT-5.2の延長線上）
2. **業界最高水準のコーディング能力**（GPT-5.3-Codexから継承）
3. **ネイティブのコンピュータ操作能力**（新規追加）

## 主な特徴

| 特徴 | 詳細 |
|------|------|
| コンテキストウィンドウ | **1,050,000トークン**（GPT-5.2の400Kから2.5倍以上） |
| 最大出力トークン | 128,000トークン |
| 入力 | テキスト + 画像 |
| 出力 | テキスト |
| Computer Use | ネイティブ対応（スクリーンショット→マウス/キーボード操作） |
| Tool Search | 47%のトークン削減を実現する新機能 |
| 推論モード | none / low / medium / high / **xhigh**（Extreme） |
| 精度向上 | 個別回答の誤り33%減、全体回答の誤り18%減（vs GPT-5.2） |
| トークン効率 | GPT-5.2比で大幅に少ないトークンで問題解決 |

## モデルバリエーション

| モデルID | 用途 | 特徴 |
|---------|------|------|
| `gpt-5.4` | 汎用 | 標準的な利用に最適 |
| `gpt-5.4-pro` | 高難度タスク | より多くの計算リソースで深く思考。Responses APIのみ |

## リリースの背景

OpenAIは「Codex GPT-5.4」として、コーディング特化モデル（GPT-5.3-Codex）と汎用推論モデル（GPT-5.2）を統合する方向へ舵を切りました。これにより、ユーザーはタスクに応じてモデルを切り替える必要がなくなりました。
